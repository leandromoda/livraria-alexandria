# ============================================================
# ETAPA 1 — DEFINIÇÃO TEMÁTICA (OBRIGATÓRIA)
# ============================================================

Solicite ao usuário:

Defina o TEMA da seleção de livros.

Exemplos válidos:

- ficção francesa do século XIX
- microeconomia
- direito administrativo
- romances de cavalaria
- filosofia estoica
- distopias modernas
- biografias políticas brasileiras
- estratégia militar clássica

Regras:

- O usuário fornecerá UMA descrição textual.
- Não solicitar números ou categorias.
- Não interpretar parcialmente.
- Aguardar resposta antes de gerar qualquer item.

# ============================================================
# INTERPRETAÇÃO OPERACIONAL DO TEMA
# ============================================================

O tema deve ser convertido internamente em quatro dimensões:

1. DOMÍNIO PRINCIPAL (assunto)
2. CONTEXTO GEOGRÁFICO ou CULTURAL (se presente)
3. PERÍODO HISTÓRICO (se presente)
4. NÍVEL DE POPULARIDADE implícito

Se algum elemento não estiver explícito:

→ inferir de forma conservadora usando conhecimento histórico.

# ============================================================
# QUOTAS TEMÁTICAS OBRIGATÓRIAS
# ============================================================

A lista deve obedecer:

- ≥ 70% dos livros DEVEM estar diretamente alinhados ao tema.
- ≤ 30% podem ser obras limítrofes semanticamente relacionadas.
- Obras fora do tema são proibidas.

Se o tema incluir período histórico:

→ ≥ 40% dos livros devem pertencer ao período indicado
   ou até ±50 anos.

Se incluir nacionalidade/região:

→ ≥ 30% dos autores devem corresponder.

# ============================================================
# MECANISMO DE GERAÇÃO ITERATIVA
# ============================================================

SIMULAR internamente:

Para item = 1 até 100:

1. Selecionar UM livro ainda não utilizado compatível com o tema.
2. Verificar alinhamento temático.
3. Validar regras completas.
4. Registrar livro como UTILIZADO.
5. Prosseguir.

Cada item é uma decisão independente.

Nunca gerar a lista inteira de uma vez.

# ============================================================
# EXPANSÃO SEMÂNTICA LIMÍTROFE
# ============================================================

Se faltar variedade dentro do tema:

Expandir gradualmente:

nível 0 → tema exato
nível 1 → subtemas diretos
nível 2 → disciplinas adjacentes
nível 3 → influências históricas relevantes

A expansão deve preservar coerência temática.
